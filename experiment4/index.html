Experiment 4 - Images, Video, & Sound Art

Imitate

To develop artistic creativity in coding, it can be helpful to start by studying and learning from existing examples. This might involve reviewing code written by others, analyzing successful techniques, or even copying and modifying code to better understand how it works. By imitating and learning from existing techniques and strategies, we can gain a better understanding of what has been done before and what is possible, and identify opportunities for improvement and innovation.

For this experiment, I didn't really have an art/code to base off to "imitate". I just had an idea where "wouldn't it be funny to have a chase scene with spongebob music playing in the background with squidward walking noise sfx". While I didn't have an "art" I wanted to imitate, I did need to find some starting points that I can start with. I googled "how to move images in p5js" and found an example that moved images according to the mouse's position. Using that examples, I imitated that so I can load an image onto p5.js and be able to move it with my mouse.

Integrate

After learning from existing techniques and strategies, we can build upon that foundation by combining elements from different sources. This might involve combining elements from different programming languages, using libraries or frameworks in new ways, or bringing together techniques from different fields or disciplines. By integrating and combining elements from different sources, we can create something new and unique.

To integrate this imitated code with my idea, I need to integrate the sounds that would play when the player moves its mouse. After adding a background image so it's not bland, I added a line of code at the Setup() function that would loop the "Grass Skirt Chase" track from spongebob. Then, in the mousePressed() function, a sound effect that soundlike like Squidward's footsteps would be played. Originally, I wanted the steps to be played when the mouse is moving, as then the footsteps would play when you are moving. However, due to the nature of the mouseMoved() function, it would've played that sfx every single frame, which created a terrible sound that played whenever the mouse was moved. I still wanted that Sfx to play in this piece, so unfortunately I had to compromise to it playing when the mouse was pressed.

Innovate

Finally, we can introduce new or original ideas to our coding process by thinking creatively and pushing the boundaries. This might involve developing entirely new techniques or approaches, or using existing techniques in new and creative ways. By innovating and introducing new ideas, we can create art that is truly groundbreaking.

Next, I had to figure out what would be "following" the slime around. While trying to figure out what should chase the slime, I was also researching how to display sound waves, as that was something I wanted to do. It was then I found the p5.Amplitude page, which showed how to use Amplitude with an ellipse. Using this page as an example, I implemented this ellipse into the drawing, and it would change its size depending on the sound(s) that is playing and would follow the slime around as the ellipse position is set to the same as the mouse. I still wanted to do a bit more with the ellipse as the color of it was pretty bland. So I added some code that would change its color each time the mouse was pressed.
