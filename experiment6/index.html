I wanted to try to use something from the lecture slides, but for some reason, I couldn't get any of the code to work on to the static webpage for some reason, despite all the settings being the same on p5js. So, I first had to think of a simple idea I wanted to do for this experiment. So, I looked around the internet to see examples of text being rotating and found examples from the documentation page form p5js here and found an example where the text was rotating to the mouse position. However, none of these examples worked with my idea of multiple text rotating. So, for the first time, I relied on ChatGPT to give me an example of the multiple images rotating in a 3D Plane. The code that I was given from ChatGPT will put 4 texts on the screen, and rotate in on the X and Y Axis every single frame. It also has an orbitControl() function, which allows players to rotate and move the camera with a mouse.

Integrate

After learning from existing techniques and strategies, we can build upon that foundation by combining elements from different sources. This might involve combining elements from different programming languages, using libraries or frameworks in new ways, or bringing together techniques from different fields or disciplines. By integrating and combining elements from different sources, we can create something new and unique.

Now, I wanted to give the users the ability to change the text that was already printed. To do this, I created an input box where players can put some kind of word or sentence into the input box. When the user then hits the button to change a text, it'll change the most left text first. After they make their first word change, if they choose to change a second word, it'll change the word to the right of the word that got changed, and it'll keep changing till it reaches the end, which then it'll loop back to the start and change the most left word again.

Innovate

Finally, we can introduce new or original ideas to our coding process by thinking creatively and pushing the boundaries. This might involve developing entirely new techniques or approaches, or using existing techniques in new and creative ways. By innovating and introducing new ideas, we can create art that is truly groundbreaking.

To innovate it, I wanted to give an option for users to add an option to add to the words that are already on to the screen, rather than changing that are already on the lists. So, I created a 2nd input box where players can type a word that they want to add. When they click on the "Add" button, it would append to the list that contains the words that are rotating on the campus, which would draw out the text that they just added and start rotating. If the word is out of the screen, users can use the mouse to position the camera in a way that they can see all the rotating words, whether by scrolling the scroll wheel to zoom out the camera or holding right click to move the camera.
